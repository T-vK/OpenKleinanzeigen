# Story 05: Local Search Agents (No Account)

## User Story

As a user without an account, I want to create local search agents that notify me when new matching items appear.

## Acceptance Criteria

- [x] CRUD UI for local search agents (name, query, filters)
- [x] Enable/disable per agent
- [x] Track seen listing IDs per agent
- [x] Notification when new listings match (configurable in settings)
- [x] Navigate: Agents tab → list → create/edit

## Definition of Done

- Agents persist across app restarts
- New-match detection unit tested
