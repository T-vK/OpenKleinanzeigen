# Story 04: Manual Search UI

## User Story

As a user without an account, I want to search for items manually with filters so I can find listings on kleinanzeigen.de.

## Acceptance Criteria

- [x] Search screen: query, location, radius, min/max price, category (optional)
- [x] Results list with image, title, price, location
- [x] Listing detail screen with description, images, link to website
- [x] Pull-to-refresh and pagination
- [x] Loading / error / empty states
- [x] Navigate from bottom bar → Search

## Definition of Done

- UI tests or Compose UI tests for search flow (mocked API)
- Manual search works against live API on device
